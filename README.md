# Customer Segmentation using K-Means Clustering

## Overview
This project implements a machine learning clustering model to group customers based on their purchasing behavior. The model identifies different customer segments which helps in understanding customer patterns and targeting marketing strategies.

## Dataset
Mall Customer Segmentation Dataset

The dataset contains customer information such as age, annual income, and spending score which are used to perform clustering.

## Objective
To apply clustering techniques to group customers into different segments based on their similarities.

## Technologies Used
- Python
- Google Colab

## Libraries Used
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Algorithm Used
K-Means Clustering

## Steps Involved
1. Import required libraries
2. Load the dataset
3. Perform data preprocessing
4. Visualize the dataset
5. Apply the Elbow Method to determine the optimal number of clusters
6. Train the K-Means clustering model
7. Visualize the clusters

## Result
The K-Means clustering algorithm successfully groups customers into different clusters based on their purchasing patterns.

## Conclusion
Customer segmentation helps businesses understand customer behavior and develop targeted marketing strategies.
